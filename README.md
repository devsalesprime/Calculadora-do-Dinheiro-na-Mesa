### Sample README.md

```markdown
# Calculadora do Dinheiro na Mesa

## Descrição

A "Calculadora do Dinheiro na Mesa" é uma ferramenta projetada para ajudar empresas a entenderem quanto estão deixando de faturar devido a processos comerciais ineficientes. Através de uma série de perguntas, a calculadora fornece uma análise detalhada da receita atual e potencial, além de insights sobre taxas de conversão e desperdícios.

## Estrutura do Projeto

- `index.html`: O arquivo HTML principal que contém a estrutura da aplicação.
- `js/`: Diretório que contém todos os scripts JavaScript necessários para a funcionalidade da calculadora.
  - `benchmark.js`: Contém dados setoriais e cálculos iniciais.
  - `validations.js`: Responsável pela validação de formulários e campos.
  - `calculations.js`: Lógica para cálculos de conversão e resultados.
  - `pdf-generation.js`: Geração de relatórios em PDF.
  - `init.js`: Inicialização do DOM e eventos principais.
- `css/`: Diretório para arquivos CSS.
  - `style.css`: Estilos personalizados para a aplicação.
- `img/`: Diretório para imagens utilizadas na aplicação.
  - `logo.png`: Marca d'água utilizada nos PDFs gerados.
- `assets/`: Diretório para ativos adicionais, como fontes e ícones.

## Como Usar

1. Clone o repositório:

   ```bash
   git clone https://github.com/devsalesprime/Calculadora-do-Dinheiro-na-Mesa.git
   ```

1. Navegue até o diretório do projeto:

   ```bash
   cd calculadora-dinheiro-na-mesa
   ```

1. Abra o arquivo `index.html` em um navegador web.

1. Siga as instruções na tela para inserir os dados da sua empresa e obter a análise.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap (para estilização)
- jsPDF e html2canvas (para geração de PDF)
